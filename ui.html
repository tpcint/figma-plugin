Integration Token

Notion 페이지 URL

사용 방법

① Notion Settings → Connections → Develop or manage integrations

② New integration 생성 → Internal Integration Secret 복사

③ 연결할 Notion 페이지 우측 상단 ··· → Connections → integration 추가

④ Token 저장 후 페이지 URL 입력 → 비교하기 클릭

💡 Token은 브라우저 로컬에만 저장되며 외부로 전송되지 않습니다.
